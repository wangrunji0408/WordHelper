### 目录内容

- `source`：所有源代码

  * `word`：Word模块，负责单词信息

  - `kernel`：Kernel模块，负责核心功能
  - `shell`：Shell模块，负责用户交互
  - `zjl_worddata`：格式化词库时所用的小程序
  - `user_data`：样例用户数据

- `resource`：所有资源文件。其中包括：

  - 格式化词库
  - 原始词库
  - 文本分析样例

- `docs`：所有文档。其中包括：

  - `design.md`：说明文档
  - `instruction.pdf`：使用说明
  - `presentation.key`：展示PPT

- `runtime`：程序运行时所需的一切文件

  按系统分别置于不同文件夹。
